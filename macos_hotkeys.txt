#### Lock
Lock
    cmd + ctrl + Q
Logout
    cmd + shift + Q
Sleep
    cmd + ctrl + S


#### Window, app
Switch to next|previous workspace
    ctrl + right|left

Focus to menu bar
    option + esc
Focus to Dock
    fn + A
Force quit menu
    cmd + option + esc
Hide Window
    cmd + H
Hide other windows
    cmd + option + H
Minimize widow (collapse to dock)
    cmd + M
Resize window
    cmd + shift + left|right|up|down
    
Close app window
    cmd + W
Quit application
    cmd + Q
    
#### Tabs
Previous tab
    cmd + option + left
Next tab
    cmd + option + right
Page down
    option + down
End
    cmd + down
    
#### Screen capture
Save full screenshot as file
    cmd + shift + 3
Copy full screenshot to clipboard
    cmd + shift + ctrl + 3
Save selected area as file
    cmd + shift + 4
Copy selected area to clipboard
    cmd + shift + ctrl + 4
Open capture menu (all capture methods)
    cmd + shift + 5
    
#### Zoom
In/out
    ctrl + scroll
    
#### Finder
Copy
    cmd + C
Paste
    cmd + V
Paste and remove source
    cmd + option + V
    
New window
    cmd + N
    
Open file
    cmd + O
Show info about file/directory
    cmd + I
Expand selected dir
    cmd + right
Open selected dir
    cmd + down
Open in new window
    cmd + dblclick
Duplicate selected
    cmd + D
Delete selected
    cmd + bacspace

Got to home directory (~/)
    cmd + shift + H
Go to ~/Desktop
    cmd + shift + D
Go to ~/Downloads
    cmd + option + L

New terminal window at selected dir
    cmd + option + R
Show/hide dotfiles
    cmd + shift + .
